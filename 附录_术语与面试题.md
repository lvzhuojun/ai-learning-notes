## 附录C：答题黄金公式 / Golden Answer Formula

### 面试答题的通用框架

```
任何AI概念，都可以用以下框架回答：

1. 是什么（What）：一句话定义
2. 为什么需要（Why）：解决了什么问题
3. 怎么做（How）：核心原理/步骤
4. 和什么对比（Compare）：与相关概念的区别
5. 优缺点（Trade-offs）：适用场景和局限
```

### 举例：用框架回答"什么是Dropout"

```
1. 是什么：
   "Dropout是一种神经网络正则化技术，训练时随机将一定比例的神经元输出置为零。"

2. 为什么：
   "为了防止过拟合——避免神经元之间形成过度依赖的协同关系。"

3. 怎么做：
   "训练时，每次前向传播随机以概率p丢弃神经元；测试时，所有神经元都激活，
   但输出乘以(1-p)保持期望值一致。"

4. 和什么对比：
   "与L1/L2正则化不同，Dropout是通过随机噪声实现正则化，
   而不是惩罚参数大小；效果上类似于训练多个子网络并集成。"

5. 优缺点：
   "优点：实现简单，效果好，计算开销小。
    缺点：增加训练时间，对小数据集效果有限，
         在RNN中使用需要特殊处理。"
```
