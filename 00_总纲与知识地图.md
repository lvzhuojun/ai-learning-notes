## 0.1 这些概念是什么关系？/ How Do These Concepts Relate?

很多人分不清 AI、ML、DL、NLP、LLM、Agent 的关系。用一个比喻来理解：

> 🏠 **AI** 是一栋大楼  
> 🧱 **ML（机器学习）** 是大楼的主体结构  
> 🔬 **DL（深度学习）** 是ML里最强大的一层  
> 🗣️ **NLP（自然语言处理）** 是DL在"语言"领域的应用  
> 🧠 **LLM（大语言模型）** 是NLP发展到今天的顶点产物  
> 🤖 **Agent（智能体）** 是给LLM装上"手脚"，让它能自主行动

## 0.2 每个领域一句话定义 / One-Line Definitions

| 领域 | 中文定义 | English Definition |
|------|---------|-------------------|
| **AI** | 让机器具备类人智能的科学与工程 | Science of making machines exhibit human-like intelligence |
| **ML** | 让机器从数据中自动学习规律，而不是手写规则 | Machines learn patterns from data without explicit programming |
| **DL** | 用多层神经网络自动提取特征的ML方法 | ML using multi-layer neural networks for automatic feature extraction |
| **NLP** | 让机器理解、生成、处理人类语言的技术 | Technology enabling machines to understand and generate human language |
| **LLM** | 在海量文本上预训练的超大规模语言模型 | Massive language models pre-trained on vast text corpora |
| **Agent** | 能自主感知环境、规划决策、使用工具完成任务的AI系统 | AI system that autonomously perceives, plans, and acts to complete tasks |

## 0.5 如何使用本教程做面试准备 / Interview Prep Guide

每个章节都遵循以下结构，针对面试有特别设计：

```
每个知识点的结构
├── 🧠 一句话理解        ← 面试开场白，直接背
├── 🍎 直觉类比          ← 帮助真正理解，说出来加分
├── 📖 正式定义          ← 准确表达，显示专业性
├── 🔗 知识关联          ← 体现系统性思维
├── 📐 核心原理          ← 深度追问时用
├── ❓ 面试常见问题       ← 直接对应真实面试题
│   └── 💡 答题框架      ← 组织语言的模板
└── ⚠️ 易错点 / 高频考点 ← 避免翻车
```

**面试答题的黄金公式**：
> **是什么（定义）** → **为什么（动机/解决什么问题）** → **怎么做（原理）** → **和什么对比（优缺点）**
